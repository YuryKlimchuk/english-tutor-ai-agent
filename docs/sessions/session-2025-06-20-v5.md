# Session 2025-06-20 (v5 — final)

## Architecture decisions (finalized)

### Bounded context relations
```
Teacher ──owns──→ Student
Teacher ──starts──→ Lesson
Lesson ──with──→ Student
Lesson ──feeds──→ Dictionary
Student ──refers──→ Dictionary (Student → DictionaryId)
Teacher ──generates──→ Exercise
Exercise ──uses──→ Dictionary
```

### Student ↔ Dictionary creation
- Use case в Teacher модуле оркестрирует:
  1. `Dictionary.api` → создать словарь → `DictionaryId`
  2. `Student.create(id, dictionaryId, ...)`
- Student знает про Dictionary, Dictionary — нет

dictionary/

### Word.java — текущее состояние
```java
@Getter @Entity @RequiredArgsConstructor
public final class Word {
    @Identity private final WordId id;
    private final String value;                  // immutable
    private final Set<String> translations;      // immutable, множество переводов
    private PartOfSpeech partOfSpeech;           // mutable (можно исправить ошибку)
    private Integer targetRepetitions;           // mutable
    private Integer currentRepetitions;          // mutable
    // статус вычисляемый: current vs target → NEW/IN_PROGRESS/LEARNED
    // конструктор/фабрика — потом
}
```
